FastAPI WebSocket 패턴 (RESEARCH.md Pattern 3):
```python
from fastapi import WebSocket, WebSocketDisconnect

@app.websocket("/ws")
async def websocket_endpoint(websocket: WebSocket):
    hub = websocket.app.state.hub
    await websocket.accept()
    await websocket.send_json(hub.get_snapshot())
    try:
        while True:
            snapshot = await hub.wait_for_change()
            await websocket.send_json(snapshot)
    except WebSocketDisconnect:
        pass
```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → /ws WebSocket | 외부 클라이언트가 WebSocket 연결 요청 |
| Browser → GET / | 외부 클라이언트가 대시보드 HTML 요청 |
| Server → Browser (WebSocket send_json) | 스냅샷 데이터가 브라우저로 전송됨 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-04 | Information Disclosure | GET /ws (스냅샷 데이터 노출) | accept | 개인용 로컬 봇. 종목 코드·현재가 노출은 허용 범위. 다중 사용자 환경 아님. Phase 7에서 봇 제어 추가 시 인증 재검토. |
| T-06-05 | Denial of Service | WebSocket 연결 무한 누적 | accept | 단일 사용자 개인 봇. 다수 연결 시나리오 없음. FastAPI의 기본 연결 관리로 충분. |
| T-06-06 | Tampering | StaticFiles 경로 순회 | mitigate | `Path(__file__).parent / "static"` 절대 경로 사용. 상대 경로 문자열 직접 사용 금지. FastAPI StaticFiles가 경로 정규화를 처리함. |
| T-06-07 | Spoofing | WebSocketDisconnect 미처리 | mitigate | except WebSocketDisconnect: pass 패턴으로 정상 종료 처리. 예외 전파 없음. |
</threat_model>
